关于
--------

本文档由 `huangz <http://huangz.me>`_ 翻译，并在原书的基础上做了以下改进：

* 修正原文正文和代码中的错误

关注项目最新进展，请到项目的 github 页面： `https://github.com/huangz1990/real-world-haskell-cn <https://github.com/huangz1990/real-world-haskell-cn>`_
